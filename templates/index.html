Monibot

This robot is designed to monitor the mine site using MQ-4, MQ-7, and DHT22 sensors.

Sensor Functions

The robot will collect methane (CH4) and carbon monoxide (CO) gas data using MQ-4 and MQ-7 sensors, as well as temperature and humidity using DHT22 sensors.

Communication

Using LoRa SX1278 to model communication between nodes and iot gateway
